1. Имя бота в телеграме Mo.Cafe
2. Перед запуском нужно поменять в файле config.py database_name на свой путь 
   и admin_telegram_id на свой id в телеграме(можно узнать с помощью бота userinfobot)
Логика системы:
	Предполагается, что у системы есть админ. Чтобы работник пользовался системой он должен сообщить свой telegram user id админу
	и он внесет его в базу(таблицу Workers).
	При первом использовании бота, он напишет работнику, что для вывода меню есть команда /menu. После её ввода выведется меню,
	которое есть в таблице Food. Также бот с меню выдаст инструкции, как надо вводить заказ(номера блюд через запятую).
	После ввода заказа, он записывается в таблицу с заказами(записывается telegram user id заказавшего, заказнные блюда и сумма заказа).
	Сумма выводится клиенту. В идеале можно сделать, чтобы этот заказ передавался в кафе, может в их телеграм или ещё куда, это несложно,
	но не стал делать, т.к. передавать некуда.
	Так же есть две команды бота специально для админа(/results_current_month и /results_previous_month). Система проверяет id админа для их выполнения.
	Они вводятся для подведения итогов месяца. Сделал два варианта, т.к. неизвестно когда будут подводится итоги, в конце текущего месяца
	или в течении следующего, с двумя вариантами будет удобнее, в зависимости от команды в таблицу с итогами записываются пользователи
	и общая сумма их заказов за текущий или предыдущий месяц. Эту сумму можно вычитать из зарплаты.
	Вычисление конечной зарплаты за вычетом заказов не стал делать, т.к. зарплата колеблется(премии и т.д.), поэтому подразумевается, что это делает админ или бухгалтер.
	Перед занесением итогов какого-либо месяца, предыдущие итоги удаляются из бд, чтобы не было уже использованной информации.
	В админке в таблице итогов есть поиск по работнику, в таблице с меню фильтр по категориям еды, в таблице работников поиск по работникам,
	в таблице заказов поиск по работнику и иерархический фильтр по дате заказа.
	Базу заполнил небольшим количеством тестовых данных, но их должно быть достаточно, чтобы увидеть как работает система.
